Future

　 The Gel that we produced has transformation quality in order to react with outside strands. This time, We controlled the gel by using only strands. But, as the future, we expect this gel will be high sensitive sensor system that react with various external conditions so that the gel acquires the reactivity for pH, pressure, temperature ...and so on. If we can equip FRET and molecular beacons on our gel, we can make them change colour and applicate it as an indicator like phenolphtalein and methyl orange. Furthermore, the fact that the pH is different around cancer cells we can diagnose cancer quickly using this gel which is sensitive to pH change. Using this gel as multiple or big, this one can cause a chain reaction like a domino. So, we can expect that we may be able to cause a signal transition as imitation of human neurotransmission In the field of molecular robots, adding this gel in some robot’s skin surface, this gel become role as the sensor device that cause some reaction in response to the outside world of the stimulus. As application, for example, delivery the gel enclosed some medicine and this one transform at location for the purpose in an organism. The gel’s simple quantity that react with some stimulus around one is expected in various situation to enhance reactivity and give quantity of florescence emission.
